🏖️ 연차계산기

입사일 기준으로 연차 발생일수와 잔여 연차를 정확히 계산하세요

✨ 주요 기능

  • 정확한 계산: 노동기준법 제60조 기준 연차 발생 계산
  • 자동 산정: 근속기간별 연차 발생일수 자동 계산
  • 사용 관리: 연차 사용 내역 추가 및 차감
  • 잔여 확인: 실시간 잔여 연차 확인
  • 휴가 계획: 연차 사용 계획 시뮬레이션

기본 정보 입력

입사일을 기준으로 연차 발생일수를 계산합니다
연차를 계산할 기준일을 입력하세요 (기본: 오늘)
근무형태에 따라 연차 발생 기준이 달라집니다
주당 근무시간 (시간제 근로자만 해당)
입력된 정보를 바탕으로 연차 발생일수를 계산합니다.

연차 사용 관리

연차를 사용한 날짜를 입력하세요
사용한 연차 일수를 입력하세요 (0.5일 단위)
연차 사용 사유를 간단히 메모할 수 있습니다
사용한 연차를 차감하여 잔여 연차를 계산합니다.

휴가 계획 시뮬레이션

휴가를 계획하는 시작일을 입력하세요
사용하려는 연차 일수를 입력하세요
입력한 일수로 휴가 계획을 시뮬레이션합니다.

🗓️ 휴가 계획 결과

📋 연차제도 안내

  • 발생 조건: 1년간 80% 이상 출근 시 15일 발생 (최초 1년 제외)
  • 최초 1년: 1개월 개근 시 1일씩 발생 (최대 11일)
  • 3년 이상: 2년마다 1일씩 추가 발생 (최대 25일)
  • 시간제 근로: 주 15시간 이상 근무 시 근무시간 비례 발생
  • 사용기한: 발생 후 1년 이내 사용 (미사용 시 소멸)
  • 분할 사용: 0.5일 단위로 분할 사용 가능
