🍼 육아휴직 계산기

  • 정확한 기간 계산 (DATEDIF 방식)
  • 분할사용 지원 (최대 3회)
  • 실시간 겹침 검증
  • 잔여기간 모의계산

  • 사용 연차 차감
  • 잔여 연차 확인
  • 연차 사용 계획 수립
