🍼 육아휴직 사용기간 계산기

육아휴직 사용기간을 정확히 계산하고 잔여기간을 확인하세요

✨ 주요 기능

  • 정확한 계산: 고용노동부 기준 DATEDIF 함수 적용
  • 분할사용 지원: 최대 3회까지 나누어 사용 가능
  • 실시간 검증: 기간 겹침 및 오류 자동 감지
  • 잔여기간 계산: 1년/1년 6개월 기준 자동 계산
  • 모의계산: 향후 사용 계획 시뮬레이션

육아휴직 사용기간 입력

1회차
기간이 겹치는 회차가 있습니다. 날짜를 확인해주세요.
입력된 기간을 바탕으로 총 사용기간과 잔여기간을 계산합니다.

잔여기간 모의계산

잔여기간을 이용한 향후 육아휴직 계획을 시뮬레이션합니다.

잔여기간 요약

💡 안내: 육아휴직은 최대 1년(12개월)까지 사용 가능하며, 아빠 육아휴직 등 특별한 경우 1년 6개월(18개월)까지 확장 가능합니다.

📅 잔여기간 상세

📋 계산 방법 안내

  • 개월수 계산: DATEDIF 함수로 시작일부터 종료일+1까지의 개월수
  • 일수 계산: 개월수로 나누어 떨어지지 않는 잔여 일수
  • 일할계산: 잔여일수를 해당월 일수로 나누어 소수점 계산
  • 합산기간: 개월수 + 일할계산값 (소수점 3자리에서 내림)
  • 1년 한도: 총 합산기간이 12개월을 초과하면 "1년 초과" 표시
  • 분할사용: 육아휴직을 여러 번 나누어 사용한 경우 회차별로 입력 (법적으로 3회까지 허용)
